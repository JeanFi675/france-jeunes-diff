# Documentation du Projet : France Jeunes Difficulté 2026

Bienvenue dans la documentation technique du site des Championnats de France Jeunes de Difficulté 2026. Ce répertoire contient toutes les informations nécessaires pour comprendre, maintenir et faire évoluer le projet.

## Sommaire

1.  **[Architecture](ARCHITECTURE.md)** : Vue d'ensemble de la pile technique, de la structure des dossiers et du flux de données.
2.  **[Composants UI](COMPONENTS.md)** : Inventaire détaillé des composants (Atoms, Blocks, Layouts) et de leur utilisation.
3.  **[Gestion des Données](DATA_MANAGEMENT.md)** : Comment mettre à jour les informations de l'événement (dates, partenaires, programme).
4.  **[Développement et Déploiement](DEVELOPMENT.md)** : Commandes utiles, workflow de travail et processus de mise en ligne.
